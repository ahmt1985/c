How to Schedule Downloads in EagleGet?

There are many scenarios that you may need the Download Scheduler function. For example, you have a plan with your ISP for a limited amount of monthly usage but with the exception of an off-peak time. EagleGet’s built-in Download Scheduler function will help you download files for the off peak time and enjoy some good sleep without worrying about exceeding your monthly usage.

There are two ways to schedule downloads.

Method 1
1. Click on the down triangle icon on Add Download dialog.

2. Choose “Schedule” under “Start with” section and then click Download to add the task.

3. If you are using this function for the first time (or if you want to change the scheduled download time), please click on the “Setting…” link, EagleGet will open the configuration window for you to set up this function.

Method 2
You can also right-click the download task to make it enter or exit the scheduled download mode.

Please Note
If you manually start a scheduled download task, the task will exit the scheduled download mode.
